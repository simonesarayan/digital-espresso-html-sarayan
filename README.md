# Architectral Strategy

# For folder structure, as the activity did not require any images in the specs, no other folders were needed. However, should there have been images required, they would have been sorted by utility. Repeating images would be stored in one folder, while page-unique images would have their own folder for organization.

# In connecting the pages, the tag <a> was utilized, along with href. As the files were all contained in one folder, there was no need to reference anything more than the file name itself.
